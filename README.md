# Wilson's Bakery Website

This project is a modern, responsive website for Wilson's Bakery, a local bakery serving Middle Georgia for over 60 years. The website is built using Next.js and styled with Tailwind CSS.

## Project Overview

Wilson's Bakery, established in 1953, is known for its famous Fingernut Cookiesâ„¢ and other delicious baked goods. This website aims to provide an improved online presence for the bakery, making it easier for customers to learn about their products.

## Features

- Responsive design for optimal viewing on all devices
- Product showcase
- About Us section detailing the bakery's history
- Contact information and location details

## Technologies Used

- [Next.js](https://nextjs.org/)
- [React](https://reactjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)

## Getting Started

To run this project locally:

1. Clone the repository
   https://github.com/kgreer0210/wilsons_bakery

2. Navigate to the project directory
   cd wilsons_bakery

3. Install dependencies
   npm install

4. Run the development server
   npm run dev

5. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
